# 股票深度技术分析工具

用于获取A股股票的专业级技术分析数据，覆盖多时间框架和多维度指标。

## 使用方法

```bash
python3 {baseDir}/scripts/technical_analysis.py <股票代码>
```

## 参数

- `股票代码`: 6位A股代码，如 `600519`（贵州茅台）、`000001`（平安银行）

## 输出

JSON 格式，包含以下模块:

### 基础数据
- `price`: 最新价、开高低收、成交量、涨跌幅、换手率
- `analysis_time`: 分析时间

### 趋势指标
- `ma`: MA5/10/20/60/120/250 均线值、方向、与价格偏离度、排列状态、金叉死叉、年线牛熊判断
- `macd`: DIF/DEA/MACD柱、金叉死叉、柱状趋势（红柱放大/缩小等）、**顶底背离检测**

### 动���指标
- `rsi`: RSI6/12/24、超买超卖状态、RSI趋势方向
- `kdj`: K/D/J值、金叉死叉钝化信号、J值极端判断
- `cci`: CCI值、超买超卖区间判断

### 波动率指标
- `bollinger`: 上中下轨、当前位置、带宽百分比、%B指标、**收口/张口判断**（突破预警）
- `atr`: ATR值、ATR占比、波动率趋势、止损参考价

### 成交量指标
- `obv`: OBV值、趋势方向、**量价背离检测**
- `volume_analysis`: 量比、放量缩量状态、量能趋势、换手率分析

### K线形态
- `candlestick_patterns`: 识别锤子线、上吊线、十字星、大阳/阴线、看涨/看跌吞没、晨星、黄昏星等形态，含可靠性评级

### 关键价位
- `levels`: 多维度支撑阻力位（近期高低点 + 均线位）
- `fibonacci`: 斐波那契回撤位（0%/23.6%/38.2%/50%/61.8%/78.6%/100%）、当前价格所在区间

### 多时间框架
- `weekly`: 周线均线排列、周线MACD信号、周线RSI

### 综合评估
- `signal_summary`: 多空信号计数、综合评分(0-100)、多空判定、置信度、所有触发信号列表

## 报告格式要求

执行脚本获取数据后，你必须严格按照 AGENTS.md 中定义的报告模板输出最终报告。报告需要：
1. 所有数字必须来自脚本输出，禁止编造
2. 对指标进行交叉验证分析，不能孤立解读单个指标
3. 结合多时间框架（日线+周线）给出趋势判断
4. 明确标注关键价位（支撑/阻力/斐波那契）
5. 给出具体的操作建议（进场区间、止损位、目标位）

## 示例

```bash
python3 {baseDir}/scripts/technical_analysis.py 600519
```
