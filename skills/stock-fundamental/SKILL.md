# 股票深度基本面分析工具

获取A股股票的专业级基本面财务数据，覆盖三张报表、估值、增长、质量检测。

## 使用方法

```bash
python3 {baseDir}/scripts/fundamental_analysis.py <股票代码>
```

## 参数

- `股票代码`: 6位A股代码，如 `600519`（贵州茅台）

## 输出

JSON 格式，包含以下模块:

### 基本信息
- `company_info`: 公司名称、行业、市值、流通市值、总股本、上市时间

### 估值数据
- `valuation`: PE（含评价）、PB（含评价）、市值、最新价、涨跌幅

### 财务指标（近8期）
- `financial_indicator`: ROE、ROA、毛利率、营业利润率、净利率、管理费用率

### 利润表（近8期）
- `profit_sheet`: 营业总收入、净利润、营业利润、营业总成本

### 资产负债表（近8期）
- `balance_sheet`: 总资产、总负债、股东权益、流动资产/负债、资产负债率、流动比率、应收账款、存货、商誉

### 现金流量表（近8期）
- `cashflow`: 经营活动现金流、投资活动现金流、筹资活动现金流

### 增长分析
- `growth`: 营收同比/环比增速、利润同比/环比增速、营收趋势、利润趋势

### 财务质量检测
- `quality_warnings`: 自动检测利润与现金流背离、高负债率、商誉风险、流动性风险、ROE下滑、毛利率异常波动

### 股东结构
- `top10_holders`: 十大股东名称、持股比例、持股数量、股东性质

## 分析流程

### 第1步：执行脚本获取数据

用 `exec` 工具执行以下命令（把 {代码} 替换为实际股票代码）：

```bash
python3 ~/.openclaw/workspace/skills/stock-fundamental/scripts/fundamental_analysis.py {代码}
```

### 第2步：深度分析 + web search 补充

**财务三表联动分析**：
- 利润表看盈利能力和趋势：营收增速、利润率变化
- 资产负债表看财务健康度：负债率、流动性、商誉风险
- 现金流量表看真实造血能力：经营现金流是否匹配利润？自由现金流是否充裕？
- 三表交叉验证：利润增长但现金流恶化 = 盈利质量下降信号

**关键比率分析**：
- 盈利能力：ROE趋势（连续>15%为优秀）、毛利率稳定性
- 成长性：营收YoY增速、利润YoY增速
- 偿债能力：资产负债率（>70%警惕）、流动比率（<1危险）
- 将比率与行业均值对比（通过 web_search 获取行业数据）

**估值判断**：
- PE/PB 与历史水平和行业均值对比
- 结合增长率判断 PEG（PE/增速）是否合理
- 高增长可以容忍高PE，低增长+高PE = 高估

**质量警告解读**：
- 认真分析 quality_warnings 中的每条警告
- 判断是一次性因素还是持续性风险

用 `web_search` 搜索补充：
- "{股票名称} 最新消息" — 近期重大事件
- "{股票名称} 行业分析" — 行业格局和竞争地位
- "{股票名称} 券商研报" — 机构观点和目标价
