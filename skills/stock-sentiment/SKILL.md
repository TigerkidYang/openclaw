# 市场深度情绪分析工具

获取A股股票的多维度市场情绪数据，覆盖资金流向、北向资金、融资融券、千股千评及综合情绪评分。

## 使用方法

```bash
python3 {baseDir}/scripts/sentiment_analysis.py <股票代码>
```

## 参数

- `股票代码`: 6位A股代码，如 `600519`（贵州茅台）

## 输出

JSON 格式，包含以下模块:

### 资金流向（个股）
- `fund_flow`: 近10日逐日数据（主力净流入、散户净流入、超大单、大单、中单）
- `fund_analysis`: 趋势分析（5日/10日累计、连续流入/流出天数、趋势方向、超大单主导性、主力散户博弈状态）

### 北向资金（大盘）
- `north_flow`: 近20日逐日北向资金净流入
- `north_analysis`: 趋势分析（5日/10日/20日累计、流入流出天数比、动量方向）

### 融资融券
- `margin_data`: 近20日融资买入额、融资余额、融券卖出量、融券余额
- `margin_analysis`: 融资余额趋势、5日变化幅度

### 千股千评
- `comments`: 综合得分、关注指数、换手率、市盈率、排名变化

### 综合情绪评分
- `sentiment`: 各维度评分（资金流35%、北向25%、融资融券15%、千股千评25%）、加权综合分(0-100)、情绪判定、情绪周期定位、所有触发信号

## 分析流程

### 第1步：执行脚本获取数据

用 `exec` 工具执行以下命令（把 {代码} 替换为实际股票代码）：

```bash
python3 ~/.openclaw/workspace/skills/stock-sentiment/scripts/sentiment_analysis.py {代码}
```

### 第2步：深度分析 + web search 补充

**资金流向深度解读**：
- 主力资金连续流入/流出天数是关键信号，连续3天以上更有意义
- 主力趋势（加速/减速/转向）比单日数据更重要
- 主力散户博弈状态："主力吸筹散户出逃"是经典底部信号，"主力出货散户接盘"是顶部信号
- 超大单主导说明机构在行动，信号更可靠

**北向资金研判**：
- 北向资金被称为"聪明钱"，其趋势对大盘和个股都有参考价值
- 关注流入动量变化（加速/减速/转向），而非单日波动
- 北向资金持续流入+个股主力流入 = 强共振信号

**融资融券信号**：
- 融资余额上升 = 杠杆做多情绪增强
- 融资余额下降 = 去杠杆，看多情绪减弱
- 融券余额大幅增加 = 做空力量增强，需警惕

**情绪周期定位**：
- 积累期：情绪温和偏多，趋势形成中 → 适合建仓
- 分布期：情绪高涨过热 → 注意风险，考虑减仓
- 恐慌期：情绪极度低迷 → 逆向思维，可能接近底部
- 调整期：情绪偏弱 → 等待��向确认

用 `web_search` 搜索补充：
- "{股票名称} 最新消息" — 近期重大事件对情绪的影响
- "{股票名称} 股吧 舆情" — 散户情绪和讨论热度
- "{股票名称} 机构调研" — 机构关注度变化
