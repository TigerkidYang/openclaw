# 市场情绪数据工具

获取A股股票的资金流向和市场情绪数据。

## 使用方法

```bash
python3 {baseDir}/scripts/sentiment_analysis.py <股票代码>
```

## 参数

- `股票代码`: 6位A股代码，如 `600519`（贵州茅台）

## 输出

JSON 格式，包含:
- `stock_name`: 股票名称
- `fund_flow`: 近10日个股资金流向（主力净流入、散户净流入、超大单等）
- `north_flow`: 近20日北向资金每日净流入
- `comments`: 千股千评数据（综合评分、关注指数等）

## 报告格式要求

执行脚本获取数据后，你必须严格按照以下格式输出最终报告：

```
# 市场情绪分析: {股票名称} ({代码})

## 资金流向
主力资金: 近5日净流入 +/-xx亿, 近10日 +/-xx亿, 趋势: 流入加速/减速/流出
散户资金: 近5日净流入 +/-xx亿, 趋势: 流入/流出

## 北向资金
近5日净流入: +/-xx亿
近20日趋势: 持续流入/流出/震荡

## 千股千评
综合得分: xx | 关注指数: xx

## 深度发现
(如果有异常发现写在这里，没有则写"无异常发现")

## 近期重要新闻
- xxx (正面/负面/中性)

## 社交媒体与舆情
整体情绪: 乐观/谨慎/悲观

## 综合情绪评分
资金流数据: xx/100 (权重60%)
舆情数据: xx/100 (权重40%)
综合: xx/100

## 结论
情绪判断: 乐观/中性/悲观
置信度: 高/中/���
总结: (一段话的专业判断)
```

## 示例

```bash
python3 {baseDir}/scripts/sentiment_analysis.py 600519
```
